# رُشد — Rushd

مركز قيادة مالي عائلي، Mobile-first وRTL، مبني على React + TypeScript + Vite.

## Sprint 01

- أول نسخة تفاعلية من الشاشة الرئيسية.
- شخصية رُشد مرسومة بالكامل بـ CSS وقابلة للحركة والتفاعل.
- صفحات: الرئيسية، حساب الشهر، الأهداف، التحليلات.
- نظام ألوان وهوية مبني على الهوية المعتمدة.
- حركة مستمرة هادئة وانتقالات Framer Motion.
- PWA manifest وتجهيز أولي للتثبيت على الجوال.

## التشغيل

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```

هذه النسخة تستخدم بيانات تجريبية. ربط Supabase والمصادقة وRealtime يأتي بعد تثبيت تجربة الواجهة الأولى.
